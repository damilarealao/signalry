CSV Format

Required Columns

  • email

Optional Columns

  • first_name
  • last_name
  • phone
  • company
  • notes
  • tags (comma-separated)

Example CSV

email,first_name,last_name,company,tags
john@example.com,John,Doe,Acme Inc.,customer,vip
jane@example.com,Jane,Smith,,"newsletter,prospect"
bob@example.com,Bob,Johnson,Startup Co.,partner

Tips

  • Make sure the CSV file has a header row with column names
  • Use UTF-8 encoding for special characters
  • You can import up to 10,000 contacts at once
  • Use double quotes for values containing commas
