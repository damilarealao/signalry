Static Groups

Static groups contain manually added contacts. You have full control over which contacts belong to these groups.

  • Manually add or remove contacts
  • Direct control over membership
  • Ideal for segmentation by behavior or preference

Dynamic Groups

Dynamic groups automatically update based on filter criteria. Contacts are added or removed automatically as they meet your conditions.

  • Automatically updated based on criteria
  • Perfect for segmentation by attributes (status, tags, etc.)
  • Saves time on manual contact management
